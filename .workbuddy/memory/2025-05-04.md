## 第二轮修改：验证码修复 + 个人主页 + 模型限制 + 公告 + APK

### 修复验证码获取失败
- **根因**: API 返回的 `captcha_id`、`svg`、`expires_in` 是顶级字段，不在 `data` 子对象中，但代码从 `result.data` 取值导致失败
- **修复**: `getCaptcha()` 改为先从顶级取，再兼容 `data` 取；ApiResult 类型新增 `captcha_id`/`svg`/`expires_in` 顶级字段
- **增强**: 注册页面增加 `captchaLoading`/`captchaError` 状态，验证码加载失败时显示错误提示和"点击刷新重试"链接

- **信息展示**: 头像（首字母）、昵称、用户名、角色 Chip
- **Token 统计**: 已使用/剩余/总额度 三栏数字展示 + ProgressBar 进度条（超过80%变红）
